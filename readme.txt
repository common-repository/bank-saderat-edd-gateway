This plugin will add Bank Parsian Iran and Shaparak Electronic Network gateway to EDD plugin

== Description ==
Easy digital downloads Bank Parsian Iran gateway

Features:

* Add Bank Parsian and Shaparak Electronic Network gateway to easy digital downloads plugin.
* Sell digital downloads through Persian WordPress in iran.

Language Support:

* Persian
* English

== Installation ==
1. Upload `bank-parsian-edd-gateway` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In the payement gateways setting of the EDD plugin put the payement gateways and default gateways on "Bank Parsian Iran"
4. complete the Merchant ID , username and password field and click save changes.
